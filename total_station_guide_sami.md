## 第一章：全站仪基础知识与萨米环境概述

### 1.1 什么是全站仪？

全站仪（Total Station），全称为"全站式电子速测仪"，是一种集光、机、电为一体的高技术测量仪器。它整合了电子测距仪（EDM）、电子经纬仪和数据处理系统，能够同时进行角度测量（水平角和垂直角）和距离测量（斜距、平距、高差），是现代工程测量和地形测绘的核心设备。

对于莱茵生命的科考工作而言，全站仪的主要用途包括：

- **建立精确的控制网：** 在冰原上确定观测站、采样点的精确坐标
- **地形测绘：** 记录冰川地貌、冰裂隙分布、地形起伏
- **变形监测：** 监测冰川移动、地壳形变、源石矿脉变化
- **工程放样：** 为临时避难所、设备安置点进行精确定位
- **考古与遗迹记录：** 测量萨米古老遗迹、洞窟壁画的位置

### 1.2 萨米的特殊环境挑战

萨米与泰拉大陆其他地区有着本质的不同。在这里使用全站仪，我们必须面对以下独特的挑战：

#### 1.2.1 极端低温

萨米冰原的气温常年处于零度以下，冬季（约泰拉历11月至次年3月）平均气温在-25°C至-40°C之间，极端情况下可达-50°C。这种低温对全站仪的影响包括：

- **电池性能急剧下降：** 锂电池在-20°C以下容量可能衰减50%以上
- **液晶显示屏响应变慢甚至冻结：** 导致读数困难或无法操作
- **机械部件收缩：** 可能导致卡滞或精度偏差
- **光学系统结霜：** 镜头和目镜结霜影响观测

#### 1.2.2 频繁的暴风雪

萨米的暴风雪（萨米语称为"*birras*"）来得毫无征兆。短短几分钟内，能见度可从数公里降至数米。暴风雪带来的问题：

- **视线遮挡：** 无法进行光学观测
- **设备积雪：** 仪器被雪覆盖，光学部件受损
- **强风威胁：** 可能导致三脚架倾倒，损坏设备
- **静电积累：** 干燥风雪环境容易产生静电，影响电子元件

#### 1.2.3 源石环境因素

作为泰拉世界的特殊存在，源石对科考设备的影响不容忽视：

- **源石辐射干扰：** 高浓度源石区域可能影响电子测距精度
- **源石粉尘：** 细小粉尘可能进入仪器缝隙，磨损精密部件
- **源石冰晶：** 特殊的源石结晶可能反射激光测距信号，造成误差

#### 1.2.4 极昼与极夜

萨米北部地区会出现极昼（夏季）和极夜（冬季）现象：

- **极昼期间：** 强烈阳光可能导致望远镜眩光，需要遮光罩
- **极夜期间：** 长时间黑暗环境下作业，需要额外照明设备

#### 1.2.5 地形复杂多变

冰原地形并非一马平川：

- **冰裂隙：** 隐藏的冰裂隙是最大威胁，设备跌落几乎必毁
- **软雪层：** 积雪深度不一，三脚架可能下陷或倾斜
- **冰面光滑：** 大风天气设备难以稳固架设

## 第二章：设备准备与检查清单

在出发前往萨米之前，充分的准备工作是确保全站仪正常工作的前提。以下是我每次冰原考察前必做的检查清单：

### 2.1 全站仪主机检查

#### 2.1.1 外观检查

- [ ] 仪器箱是否完好，锁扣、提手牢固
- [ ] 仪器表面无明显划痕、凹陷
- [ ] 望远镜物镜、目镜清洁无霉斑
- [ ] 基座连接器无磨损、松动
- [ ] 各旋钮、制动螺丝操作顺畅
- [ ] 数据接口（USB/RS232）保护盖完好

#### 2.1.2 功能测试（室内）

在温暖的室内进行以下测试：

- [ ] 开机自检正常通过
- [ ] 水平角、垂直角读数正常
- [ ] 电子气泡居中功能正常
- [ ] 激光对中器工作正常（如有）
- [ ] 测距功能测试（短距离）
- [ ] 数据存储/读取功能正常
- [ ] 触摸屏/按键响应灵敏
- [ ] 内置软件版本确认

#### 2.1.3 精度验证

使用室内基线或短距离标准基线进行：

- [ ] 角度重复性测试（盘左盘右各测3次）
- [ ] 测距精度测试（与标准尺比对）
- [ ] 补偿器功能验证

### 2.2 配件准备

#### 2.2.1 三脚架

- [ ] 检查架腿伸缩顺畅，锁紧装置可靠
- [ ] 架头无松动，中心螺丝适配仪器基座

#### 2.2.2 棱镜与目标

### 2.3 个人防护装备

- [ ] 防风保暖手套（指尖可触屏型，方便操作）

## 第三章：全站仪的安装与架设

在萨米冰原上架设全站仪，比在普通环境下需要更多技巧和注意事项。一个稳固、精确的架设是获得可靠数据的基础。

### 3.1 选址原则

#### 3.1.1 安全性优先

- **远离冰裂隙：** 使用冰锥或探杆探测架设点周围至少5米范围内的冰层厚度
- **避开雪崩区：** 观察上方地形，避免在陡坡下方作业
- **远离野生动物活动区：** 萨米有独特的生态系统，大型动物可能误碰设备
- **考虑退路：** 确保在恶劣天气下可以安全撤离

#### 3.1.2 观测条件

- **通视条件：** 确保与目标点之间无遮挡
- **避免强反射面：** 冰面、光滑岩壁可能产生反射干扰
- **远离热源：** 营地火炉、热泉等会造成空气折射
- **避开强磁场源：** 大型金属设备、源石矿脉可能影响指北精度

#### 3.1.3 稳定性考虑

- **坚实地面：** 优先选择岩石露头、压实冰面
- **避开软雪区：** 积雪深度超过30厘米的区域不适合架设
- **背风面：** 如可能，选择天然屏障（岩石、冰丘）背风侧

### 3.2 三脚架架设

#### 3.2.1 普通积雪地面

1. **清理积雪：** 用脚或雪铲清除直径约1米的圆形区域，露出坚实冰面或压实雪层
2. **架设三脚架：**
   - 张开架腿至合适角度（约60°）
   - 确保架头大致水平
   - 架腿均匀受力，避免单腿悬空
3. **加固措施：**
   - 在架腿周围踩踏压实雪层
   - 可在架腿底部放置垫板增加受力面积
   - 使用雪锚或冰镐固定（强风环境）

#### 3.2.2 光滑冰面

冰面是全站仪架设最危险的场景之一：

1. **使用冰爪：** 在三脚架脚底部安装金属冰爪
2. **增加摩擦：** 在架腿下垫防滑垫或橡胶垫片
3. **配重固定：** 在三脚架中盘悬挂重物（背包、石头）增加稳定性
4. **绳索牵引：** 必要时用绳索将三脚架固定在冰镐或雪锚上

#### 3.2.3 陡坡地带

- 两架腿朝向下坡方向，一架腿朝向上坡
- 调整架腿长度使架头尽可能水平
- 使用较长的冰镐在上坡方向做额外支撑

### 3.3 仪器安装与对中整平

#### 3.3.1 安装步骤

1. **取出仪器：**
   - 在仪器箱内完成初步除霜（如果箱子从温暖环境带出）
   - 检查仪器表面无积雪、冰晶
   - 确认基座连接部位清洁

2. **安装到三脚架：**
   - 一手握住仪器提手，另一手托住基座
   - 将仪器轻放在三脚架头上，对准中心螺丝
   - 旋紧中心螺丝，用手旋紧即可，**严禁使用工具暴力拧紧！**

3. **初始对中：**
   - 使用光学对中器或激光对中器
   - 调整三脚架架腿位置，使对中器大致对准地面标志点
   - 此时架头不需要完全水平，粗略水平即可

#### 3.3.2 精确整平与对中

全站仪的整平与对中是一个需要耐心的迭代过程：

**步骤一：粗略整平**
- 转动仪器，使长水准器与任意两个脚螺旋连线平行
- 双手同时向内或向外旋转这两个脚螺旋，使气泡居中
- 转动仪器90°，调整第三个脚螺旋使气泡居中

**步骤二：精确对中**
- 观察光学/激光对中器，检查是否对准地面点
- 如偏差较大，松开中心螺丝，平移仪器基座
- 如偏差较小，调节三脚架架腿高度（伸缩式）或整体移动三脚架

**步骤三：精确整平**
- 重复步骤一的整平操作，使用电子气泡进行精平
- 现代全站仪的电子补偿器可以自动修正微小倾斜，但仍应尽量整平

**步骤四：反复检查**
- 再次检查对中，如有偏差重复上述步骤
- 目标：对中误差 ≤ 1mm，整平误差 ≤ 1′

> **麦哲伦的警告：** 在萨米，地面点可能被雪掩埋。建议：
> - 使用醒目的标志（彩色标志旗、荧光棒）
> - 在标志点周围清理积雪
> - 拍摄照片记录标志点相对于周边地物的位置

### 3.4 开机预热与环境适应

仪器从温暖的帐篷或背包中取出后，需要时间适应环境温度：

1. **静置适应：** 架设完成后，让仪器在环境中静置5-10分钟
2. **开机自检：** 按下电源键，等待自检程序完成
3. **观察状态：** 注意屏幕显示、电池电量、温度读数
4. **初步测试：** 进行简单的角度测量，确认仪器响应正常

## 第四章：基础测量操作

### 4.1 角度测量

#### 4.1.1 水平角测量原理

水平角是两目标方向线在水平面上的投影所夹的角度。全站仪通过水平度盘和读数系统自动完成测量。

**盘左（正镜）与盘右（倒镜）：**
- 盘左：竖直度盘位于望远镜左侧时的观测位置
- 盘右：竖直度盘位于望远镜右侧时的观测位置
- 通过盘左盘右观测取平均，可以消除视准轴误差、横轴误差等系统误差

#### 4.1.2 测回法测量水平角

这是最基本的水平角测量方法：

**步骤一：盘左观测**
1. 瞄准目标A（后视），读取水平角读数 L_A
2. 顺时针旋转仪器瞄准目标B（前视），读取水平角读数 L_B
3. 计算上半测回角值：β_L = L_B - L_A

**步骤二：盘右观测**
1. 倒转望远镜（盘右位置）
2. 瞄准目标B，读取水平角读数 R_B
3. 逆时针旋转仪器瞄准目标A，读取水平角读数 R_A
4. 计算下半测回角值：β_R = R_B - R_A

**步骤三：计算结果**
- 一测回角值：β = (β_L + β_R) / 2
- 半测回差应满足规范要求（如DJ2级仪器≤12″）

#### 4.1.3 方向观测法（全圆测回法）

当需要观测多个方向时，使用方向观测法：

1. 盘左位置，依次瞄准目标A、B、C、D、A（归零）
2. 盘右位置，依次瞄准目标A、D、C、B、A（归零）
3. 计算各方向的平均值
4. 进行方向归零、2C差、测回差等检核

#### 4.1.4 垂直角测量

垂直角是目标方向与水平面之间的夹角。

**观测步骤：**
1. 盘左位置，十字丝横丝切准目标顶部
2. 读取垂直角读数（天顶距或高度角，依设置而定）
3. 盘右位置，再次切准同一目标
4. 读取垂直角读数
5. 取盘左盘右平均值

**指标差检核：**
- 理论上，盘左读数 + 盘右读数 = 360°（天顶距模式）
- 如果不等，其差值的一半为指标差
- 指标差应在允许范围内（如DJ2级仪器≤15″）

### 4.2 距离测量

#### 4.2.1 电子测距原理

全站仪的EDM（电子测距仪）通过发射调制光波（红外或激光），测量光波往返目标的相位差或时间差，计算距离。

**斜距、平距、高差的关系：**
- 斜距（S）：仪器中心到目标棱镜中心的直线距离
- 平距（H）：水平面上的投影距离，H = S × cos(α)
- 高差（V）：垂直方向上的差值，V = S × sin(α)
- 其中α为垂直角

#### 4.2.2 测距操作步骤

1. **参数设置：**
   - 输入气象参数（温度、气压，萨米标准大气压约1013hPa）
   - 选择棱镜常数（常用-30mm，务必确认与所用棱镜一致）
   - 设置测距模式（精测/粗测/跟踪）

2. **瞄准棱镜：**
   - 使用望远镜十字丝中心对准棱镜中心
   - 确保光路畅通，无遮挡
   - 检查信号强度指示，确保足够

3. **启动测量：**
   - 按下测距键，等待结果显示
   - 通常需要进行2-3次读数取平均
   - 记录斜距、平距、高差

#### 4.2.3 测距精度影响因素

在萨米环境，需特别注意：

- **大气折射：** 冰原上空温度梯度大，光线折射明显
- **雪雾影响：** 空气中冰晶可能散射光信号
- **棱镜温度：** 极低温可能影响棱镜光学特性
- **振动干扰：** 强风引起的仪器振动

### 4.3 坐标测量

#### 4.3.1 坐标系统简介

全站仪测量需要建立坐标系统：

- **测站坐标（N₀, E₀, Z₀）：** 仪器所在位置的坐标
- **后视方向：** 确定坐标北方向的参考方向
- **定向角：** 后视方向相对于坐标北的方位角

#### 4.3.2 测站设置步骤

1. **输入测站坐标：**
   - 通过键盘输入已知的测站坐标
   - 或使用自由设站程序计算

2. **后视定向：**
   - 输入后视点坐标，或输入后视方位角
   - 瞄准后视棱镜，确认定向
   - 建议盘左盘右各定向一次，取平均

3. **检查定向：**
   - 测量另一个已知点作为检查
   - 比较测量坐标与已知坐标的差异
   - 偏差应在允许范围内（如≤5mm+5ppm）

#### 4.3.3 碎部测量

完成设站后，即可测量任意目标点的坐标：

1. 瞄准目标棱镜
2. 启动测量
3. 记录显示的N、E、Z坐标
4. 可同时记录点号、编码、属性等信息

### 4.4 放样测量

放样是根据设计坐标，在现场标定出相应位置的过程。

#### 4.4.1 坐标放样步骤

1. **设站定向：** 同4.3.2
2. **输入放样点坐标：** 输入目标点的设计坐标
3. **计算放样数据：** 仪器自动计算距离差（ΔD）和角度差（Δβ）
4. **角度放样：**
   - 转动仪器，使水平角差值为0°00′00″
   - 指挥持镜者在视线方向上移动
5. **距离放样：**
   - 测量实际距离，与理论距离比较
   - 指挥持镜者前后移动，直至距离差为0
6. **精确定位：** 反复调整角度和距离，直至达到精度要求

#### 4.4.2 极坐标放样

适用于快速放样多个点：

1. 计算各放样点相对于测站的极坐标（距离和方位角）
2. 依次瞄准各方向，测量距离定位

#### 4.4.3 萨米放样注意事项

- **沟通困难：** 暴风雪中视线和通讯受限，建议使用预设信号或对讲机
- **移动困难：** 冰原上行进缓慢，提前规划路线
- **标志保存：** 使用醒目的标志物，并拍照记录相对于固定地物的位置

## 第五章：特殊环境下的操作技巧

### 5.1 极低温环境下的操作

#### 5.1.1 设备预热与保温

- **仪器保温：** 使用保温罩包裹仪器，仅露出望远镜和操作面板
- **电池保温：** 轮换使用贴身保温的电池，避免长时间暴露在低温中
- **预热时间：** 开机后等待仪器内部温度稳定（约5-10分钟）
- **避免急冷急热：** 不要将温暖的仪器直接暴露在极寒空气中，应逐渐适应

#### 5.1.2 屏幕与操作

- **屏幕保护：** 使用防眩光贴膜，极昼时减少阳光反射
- **除霜措施：** 准备小毛刷和镜头纸，及时清除屏幕和目镜霜花
- **操作节奏：** 低温下手套操作不便，预留更充足的操作时间
- **简化操作：** 提前设置好常用程序，减少现场输入

#### 5.1.3 观测技巧

- **缩短测程：** 低温下大气折射不稳定，适当缩短单站观测距离
- **增加测回数：** 适当增加测回数以抵消环境不稳定带来的误差
- **避开温度梯度大时段：** 日出后和日落前温度变化剧烈，尽量避免精密测量

### 5.2 暴风雪天气应对

#### 5.2.1 预警与撤离

- **关注天气：** 使用便携气象站或接收天气预报
- **预警指标：**
  - 能见度开始下降
  - 风速明显增大
  - 气压快速下降
- **撤离原则：** 一旦收到暴风雪预警，立即撤离，不要犹豫！

#### 5.2.2 紧急保护

如暴风雪突然来袭，来不及撤离：

1. **立即关机：** 保护电子设备
2. **覆盖防护：** 使用防雪罩或背包覆盖仪器
3. **加固三脚架：** 如有绳索，立即固定
4. **人员避险：** 寻找背风处或搭建临时掩体，人员安全优先

#### 5.2.3 暴雪后恢复

- **清理积雪：** 使用软毛刷轻轻清除积雪，**严禁敲打仪器！**
- **检查稳定性：** 检查三脚架是否下陷或松动
- **除霜除冰：** 用镜头纸和无水乙醇清洁光学部件
- **功能检查：** 开机自检，确认各项指标正常
- **数据备份：** 立即导出已测数据，防止后续故障丢失

### 5.3 源石区域作业

#### 5.3.1 辐射监测

- 使用便携式源石辐射检测仪监测环境辐射水平
- 避免在高浓度源石区域长时间停留
- 佩戴个人防护装备（如适用）

#### 5.3.2 测量影响

- **测距误差：** 源石结晶可能反射或散射测距信号，注意观察数据异常
- **磁干扰：** 某些大型源石矿脉可能影响电子罗盘或磁补偿器
- **数据记录：** 详细记录源石分布情况，以便后续数据处理时考虑

#### 5.3.3 设备保护

- 作业后及时清洁仪器表面，防止源石粉尘附着
- 使用防静电措施，避免静电损伤电子元件
- 定期检查仪器密封性，防止粉尘进入

### 5.4 极昼与极夜作业

#### 5.4.1 极昼作业

- **遮阳措施：** 使用望远镜遮光罩，减少眩光
- **人员防护：** 佩戴防紫外线护目镜和防晒霜
- **时间感：** 连续白昼容易疲劳，严格遵守作息时间表

#### 5.4.2 极夜作业

- **照明设备：** 准备充足的人工照明
- **棱镜照明：** 使用反光材料或LED灯照亮棱镜，便于瞄准
- **供电保障：** 照明增加电力消耗，准备更多备用电池
- **安全措施：** 黑暗中移动危险，使用绳索连接队友，携带信号棒

## 第六章：数据管理与记录

### 6.1 数据采集规范

#### 6.1.1 记录内容

每次测量应记录：

- **测站信息：** 测站编号、坐标、仪器高
- **观测条件：** 日期时间、天气、温度、气压、能见度
- **仪器信息：** 仪器型号、编号、棱镜常数
- **观测数据：** 盘左盘右读数、测回编号
- **点号编码：** 清晰、唯一的点号，有意义的编码
- **备注：** 特殊情况、异常现象、照片编号

#### 6.1.2 编码系统建议

建立统一的编码系统，例如：

- `CZ001` - 测站（测站拼音首字母）
- `FS001` - 分水准点
- `KZ001` - 控制点
- `XB001` - 雪边界点
- `LX001` - 裂隙点
- `JY001` - 基岩点

### 6.2 数据导出与备份

#### 6.2.1 日常备份

- **当日数据当日导出：** 不要等存储卡满了才导出
- **双备份原则：** 至少保存两份数据（如手簿+电脑，或两张存储卡）
- **格式转换：** 导出为通用格式（CSV、TXT、DXF等）
- **元数据保存：** 同时保存仪器设置、测站信息等元数据

#### 6.2.2 纸质记录

在萨米，永远不要完全依赖电子设备！

- 准备防水记录本
- 使用铅笔记录（圆珠笔墨水会冻结，钢笔会漏墨）
- 记录关键数据作为电子数据的备份和检核
- 绘制草图，标注测量点位置

### 6.3 数据处理流程

1. **数据导入：** 将原始数据导入处理软件（如徕卡LGO、南方CASS等）
2. **数据检核：** 检查超限观测、计算闭合差
3. **平差计算：** 进行网平差，得到最或是坐标
4. **精度评定：** 计算单位权中误差、点位中误差
5. **成果输出：** 生成坐标表、图纸、报告

## 第七章：维护保养与故障排除

### 7.1 日常维护

#### 7.1.1 每日作业后

- **清洁：** 使用软毛刷和镜头纸清洁仪器表面和光学部件
- **干燥：** 将仪器放入干燥剂保存箱，防止内部结露
- **检查：** 检查电池电量，及时充电
- **记录：** 记录当日使用情况和异常情况

#### 7.1.2 定期维护（每周）

- 检查三脚架螺丝紧固情况
- 清洁数据接口，检查接触是否良好
- 检查棱镜和棱镜杆状态
- 备份所有数据到外部存储

### 7.2 极寒环境特殊保养

#### 7.2.1 温度适应

- **避免温度冲击：** 不要将冰冷的仪器直接带入温暖的帐篷，应先放入仪器箱，让温度缓慢上升
- **防止结露：** 温度上升过程中，仪器表面可能结露，保持箱内干燥剂充足
- **彻底干燥：** 回到营地后，彻底干燥仪器和配件

#### 7.2.2 润滑保养

- 极低温可能使润滑油凝固，影响机械部件
- 如有条件，使用低温专用润滑油
- 定期操作各旋钮，保持灵活性

#### 7.2.3 电池保养

- 不要在极低温下充电
- 长期不用时，电池应半电量保存
- 定期检查电池健康状态

### 7.3 常见故障与处理

#### 7.3.1 无法开机

**可能原因：**
- 电池电量耗尽或低温失效
- 电池接触不良
- 内部故障

**处理步骤：**
1. 更换贴身保温的备用电池
2. 检查电池触点是否清洁
3. 如仍无法开机，联系技术支持

#### 7.3.2 测距失败或异常

**可能原因：**
- 光路被遮挡
- 信号太弱（距离过远或棱镜问题）
- 大气条件恶劣
- 仪器参数设置错误

**处理步骤：**
1. 检查望远镜与棱镜之间的通视
2. 检查棱镜是否清洁、对准
3. 检查棱镜常数设置
4. 缩短距离测试，确认仪器正常
5. 等待天气好转

#### 7.3.3 角度测量异常

**可能原因：**
- 补偿器故障
- 度盘问题
- 仪器未整平

**处理步骤：**
1. 重新整平仪器
2. 重启仪器，观察自检结果
3. 进行盘左盘右观测，比较2C值
4. 如问题持续，停止使用，送修

#### 7.3.4 屏幕显示异常

**可能原因：**
- 低温导致液晶响应变慢
- 屏幕结冰或结霜
- 背光故障

**处理步骤：**
1. 使用保温罩或暖宝宝加温
2. 清除屏幕霜花
3. 调整对比度设置
4. 如无法恢复，使用外接手簿

### 7.4 长期存储

如需长期存放全站仪：

- 彻底清洁并干燥
- 取出电池单独存放
- 存放于干燥、通风、恒温环境
- 定期检查（每3个月通电检查一次）
- 三脚架松开锁紧装置，防止弹簧疲劳

## 第八章：安全与应急

### 8.1 人员安全

#### 8.1.1 个人防护

- **永远不要单独作业！** 至少两人一组
- 携带卫星电话或紧急信标
- 随身携带急救包
- 了解冻伤、雪盲等极地疾病的症状和处理方法
- 严格遵守作息，防止疲劳作业

#### 8.1.2 通讯保障

- 建立定期联络制度（如每小时报平安）
- 约定紧急信号和集合点
- 携带备用通讯设备和电池

### 8.2 设备安全

#### 8.2.1 防盗防损

- 作业时仪器不离开视线
- 休息时收起放入帐篷
- 营地设置警戒，防止野生动物损坏

#### 8.2.2 应急撤离

如需紧急撤离：

1. 优先保障人员安全
2. 如时间允许，收起仪器
3. 如情况紧急，放弃设备，人员立即撤离
4. 记录撤离位置，事后尝试找回

### 8.3 环保责任

- 遵守萨米当地法规和文化传统
- 不破坏自然环境和野生动物栖息地
- 带走所有垃圾，包括废弃电池
- 尊重萨米原住民的文化遗产
