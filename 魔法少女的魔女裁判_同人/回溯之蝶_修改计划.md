# 《回溯之蝶》修改计划

## 当前问题

梅露露在结尾突然出现并自杀，缺乏前文铺垫，读者会感到困惑。

## 修改目标

在不改变原有情节走向的前提下，为梅露露的出场添加足够的伏笔，让结尾的牺牲有情感重量。

## 具体修改方案

### 1. 第二章结尾处添加伏笔

在月代雪说完"有些命运，是逃不掉的"之后，希罗的笔记本内容里加入一段：

```
我合上笔记本，却注意到走廊尽头有一个身影。
银白色的长发，在月光下像是一层薄霜。她在那里站了多久？
当我追出去时，走廊已经空了。
只有空气中残留的一丝...消毒水的气味？
```

### 2. 第三章中段添加线索

在希罗跟踪月代雪到祭坛时，添加环境描写：

```
祭坛周围散落着一些物品——不是仪式用具，而是日常生活的东西。
一本翻到一半的书，几个用过的茶杯，还有...
一件叠得整整齐齐的银色外套。
像是有人长期住在这里。
```

### 3. 第三章结尾处强化暗示

月代雪说完"我找到了"之后，希罗追问：

```
"你说的'找到了'...除了我，还有谁？"

月代雪的表情变得柔软，那是我第一次看到她露出那样的神情。
"还有一个笨蛋。"她轻声说，"一直在找我，找了很久很久的笨蛋。"
```

### 4. 第四章审判场景添加一笔

在最终审判开始时，希罗俯瞰众人时注意到：

```
十三名同伴...不对，是十四道身影。
在祭坛的阴影处，站着那个我在走廊见过的银发少女。
她没有抬头，只是静静地看着月代雪，眼神里是全然的信任。
```

### 5. 尾声前添加回忆片段（可选）

在梅露露冲出来之前，插入一段希罗的回溯画面：

```
在无数条时间线中，我曾瞥见过这一幕——
那个银发少女，在不同的轮回里，始终站在月代雪身后。
她操纵着监狱的守卫，策划着每一场审判，只为找到让月代雪"回家"的方法。
五百年的执念，终于在这一刻有了出口。
```

## 修改原则

- 不改变原有情节走向
- 不增加新角色，只为梅露露的出场做铺垫
- 保持希罗的第一人称视角
- 伏笔要轻，不能抢主线风头

## 预期效果

读者在结尾看到梅露露时，会意识到"原来前面那些暗示都是她"，而不是"这人是谁"。牺牲的重量来自于认知的建立。
